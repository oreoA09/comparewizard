How to use Compare Wizard

1

Upload the file for analysis or comparison

2

Choose the function you wish to perform
“Analyze” button to highlight the duplicates in one file,
“Compare” button to run a comparison betweeen two files and highlight the duplicates,
“Compare and fix” to run a comparison betweeen two files, highlight and remove duplicates within them and return both files,
“Compare edit and merge” to run a comparison betweeen two files, highlight and remove duplicates within them and merge both files into one.

3

You can now choose to save, publish to community and download your files in different formats.
